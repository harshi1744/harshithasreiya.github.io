⚙️ Skills Demonstrated

  • Resume Parsing & NLP
  • Semantic Similarity & Hybrid Scoring
  • Conversational AI (LLM Integration)
  • Agentic System Design
  • Feedback-Driven Learning Loop
  • Privacy-Centric Architecture

Python 3.x, Streamlit, PyMuPDF, SentenceTransformers, Pandas, ctransformers (Mistral-7B), RAG Architecture

  • 🔐 Offline Mode: Fully local operation, no cloud

📈 Impact & Outcomes

  • ⏱️ Reduced resume triage time
  • 🌐 Transparent scoring and rationale
  • 🔄 Dynamic adaptation via recruiter feedback
  • 🛡️ Designed for strict data privacy
