🤖 Hiring Assistant AI – Agentic Resume Screening System

To revolutionize traditional hiring by building a lightweight, privacy-compliant, and offline-ready AI assistant that autonomously screens resumes, ranks candidates, and interacts dynamically with recruiters—powered by Retrieval-Augmented Generation (RAG) and continual feedback adaptation.

⚙️ Skills Demonstrated

  • Agentic AI Design & Prompt Engineering
  • Retrieval-Augmented Generation (RAG)
  • Semantic Similarity & Embedding Techniques
  • Feedback-Driven Adaptation
  • Privacy-Centric Architecture
  • Streamlit App Development
  • Human-AI Interaction Design

🛠️ Tools & Technologies

Python 3.x | PyMuPDF | SentenceTransformers (MiniLM-L6-v2) | Pandas | Streamlit | ctransformers (Mistral-7B-Instruct) | In-Memory Vector Search | RAG | Offline Inference Framework

🔑 Core Features & Workflow

  • 📄 Resume Parsing: Extracts structured content from resumes & job descriptions using PyMuPDF
  • 🔍 Semantic Matching: Uses SentenceTransformers to embed and compare candidate/job text
  • 🧮 Hybrid Scoring: Combines semantic similarity with rubric-based skill matching using Pandas
  • 💬 Recruiter Chat UI: Streamlit interface backed by a quantized Mistral-7B-Instruct model
  • 🔄 Adaptive Feedback Loop: Learns recruiter preferences in real time
  • 🔐 Offline & Secure: Fully local, privacy-focused operation with no cloud dependencies

📈 Key Outcomes

  • ⬇️ Time-to-Hire: Significantly reduced screening time through automation
  • 📊 Transparent Matching: Clear score explanations improved recruiter trust
  • ⚙️ Evolving System: Adapted scoring and ranking via recruiter feedback
  • 🔐 Fully Private: Local, GDPR-compliant architecture ensured data sovereignty

🔎 Outcome

An intelligent, privacy-first hiring assistant that enhances recruiter efficiency, fairness, and decision transparency through adaptive, agentic AI.

🧠 Reflection

"This project deepened my understanding of building secure, agentic AI tools that respect user privacy while enhancing decision quality. Designing a transparent, feedback-driven system taught me the balance between automation and human oversight in sensitive applications like hiring."

  • Real-Time Data Processing & Alerting
  • Model Explainability (SHAP, LIME)
  • Ethical & Fair AI Design

Python | Scikit-learn | XGBoost | SHAP | LIME | Kafka | Docker | Flask | PostgreSQL | Prometheus | Grafana | AES-256 | TLS 1.3 | Role-Based Access Control (RBAC)
