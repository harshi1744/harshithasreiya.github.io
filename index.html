2. Visualizing Global Trends in Child Malnutrition (Academic Project)

Leveraging global health datasets from WHO, UNICEF, and the World Bank, this project offered a deep dive into malnutrition trends—analyzing stunting, wasting, underweight, and overweight metrics across geography, gender, and age. By transforming complex raw data into interactive dashboards, the project empowered global stakeholders to make informed decisions targeting child health inequalities.

Key Deliverables:

  • Malnutrition Severity Dashboard: Filterable by country, region, gender, and year—offering a 360° view of malnutrition prevalence
  • Child Patterns Dashboard: Trend analysis by age, visualized through scatter plots, time-series, box plots, and stacked area charts
  • Geospatial Heatmaps: Interactive mapping to highlight regional malnutrition disparities

3. Financial Fraud Detection — Machine Learning Model (Academic Project)

This project addressed the escalating challenge of financial fraud using supervised machine learning. By applying multiple models on a labeled dataset, the system aimed to detect suspicious transactions effectively, prioritizing recall to minimize undetected fraud and integrating fairness metrics to reduce bias.

Approach & Methodology:

  • Dataset: Publicly available transaction data with fraud labels
  • Preprocessing: Balanced classes using SMOTE, handled missing values, feature scaling
  • Models Trained: Logistic Regression, Random Forest, XGBoost, Neural Networks
  • Evaluation Metrics: Precision, Recall, F1-score, ROC-AUC, with focus on minimizing false negatives
  • Fairness Checks: Monitored model bias across demographics
